# Sortu — Product Brief

## Masalah

Gaji masuk → campur aduk. Saat tagihan listrik / Netflix / top-up game datang, user harus:

- hitung ulang sisa uang
- transfer bolak-balik antar rekening/e-wallet
- lupa apakah “jatah Netflix” bulan ini sudah disisihkan

## Solusi

Sortu = **amplop digital**:

1. Masukkan pemasukan (tercatat)
2. Sortir ke kantong (listrik, Netflix, game, …)
3. Pantau progress sampai jatuh tempo
4. Saat bayar: satu tap alur → saldo kantong berkurang + tercatat (fase 1: konfirmasi manual; fase 2: payment gateway)

## Target user

- Anak muda / pekerja yang punya 3+ langganan atau tagihan rutin
- Sudah pakai e-wallet, tapi ingin **disiplin alokasi**, bukan hanya tracking belanja

## Differentiator vs kompetitor

| Kompetitor | Mereka | Sortu |
|------------|--------|-------|
| Goodbudget / YNAB | Envelope global, UI berat / berbayar | Ringan, ID-first |
| Jatahku | Envelope + Telegram NLP | App visual fokus kantong & jatuh tempo |
| Jenius pockets | Butuh bank Jenius | Independen, di atas uang yang sudah dimiliki user |
| Money tracker biasa | Catat setelah belanja | **Rencana dulu**, belanja/bayar kemudian |

## Monetisasi (belakangan)

- Free: 5–7 kantong
- Premium: unlimited + reminder + export
- Atau jual source sebagai produk siap rebrand (Freelance)

## Non-goals (MVP)

- Menjadi bank / menyimpan dana user
- Integrasi API Netflix / PLN langsung
- Multi-currency kompleks
